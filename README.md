# bdt-mini-project
- Install Kafka and Zookeeper Docker Images
- Install Hadoop on your system
- Install the following Python Libraries:
	- flask
	- pymongo
	- kafka
	- sklearn
	- librosa
	- numpy
	- flask_cors

- Run the Kafka and Zookeeper images as containers
- Start Hadoop
- Run the app using the command: python app.py
